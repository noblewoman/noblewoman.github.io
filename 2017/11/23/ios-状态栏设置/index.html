ios 状态栏设置

设置分为两种情况，一种是全局隐藏，一种是局部隐藏。

全局隐藏： target ->Deployment Info -> Status Bar style -> hidden status bar。
局部隐藏分为两种情况：
一：

info.plist 文件中View controller-based status bar appearance的设置是YES。说明viewController对status的设置优先级高于application。（如果没有这个配置的话，需要自己手动添加）
其次：在对应VC中调用
[self prefersStatusBarHidden];

-(BOOL)prefersStatusBarHidden{
return YES;
}
二：

如果 View controller-based status bar appearance 设置为NO，则上述方法调用则失效，可调用
隐藏掉状态栏
[[UIApplication sharedApplication] setStatusBarHidden:YES];
展示状态栏：
[[UIApplication sharedApplication] setStatusBarHidden:NO];
